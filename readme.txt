== Description ==

Customize settings using the settings API & customize WordPress Help messages. This is a temporary project as these
features were recently removed from the NOUVEAU theme core and need to be organized.

This example is part of the NOUVEAU WordPress framework. For more great starting code, search for "Nouveau" in
WordPress.org's plugins or visit [www.nouveauframework.com]

== Installation ==

1. Install using your WordPress sites admin (under Plugins &rarr; Add New), or by uploading the plugin folder to your `/wp-content/plugins/` directory
2. Activate the plugin from your sites admin (under Plugins)
